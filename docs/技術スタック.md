# ClariFi 技術スタック

最終更新: 2026-05-17

---

## 確定技術スタック

| 役割 | 技術 | 理由 |
|------|------|------|
| フロントエンド | Next.js 14 + TypeScript | ポートフォリオ映え・SSR対応・拡張性◎ |
| バックエンド | Node.js + Express + TypeScript | 分離構成・本格的なAPI設計 |
| スクレイピング | Playwright (Node.js) | WAON自動取得・Windows対応◎ |
| データベース | PostgreSQL | 月次・カテゴリ集計クエリに最適 |
| ORM | Prisma | TypeScript親和性◎・型安全 |
| グラフ | Recharts | React/Next.js親和性◎ |
| 定期実行 | node-cron | Express内で完結 |
| 認証情報管理 | dotenv | ローカル環境変数で安全管理 |
| パッケージ管理 | pnpm (モノレポ) | モノレポ管理しやすい |

---
