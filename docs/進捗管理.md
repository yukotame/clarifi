### Phase 1-A 実行コマンド手順

```bash
# Step 1: pnpmインストール（未インストールの場合）
npm install -g pnpm

# Step 2: リポジトリをクローン
git clone https://github.com/yukotame/clarifi.git
cd clarifi

# Step 3: 全パッケージインストール
pnpm install

# Step 4: .envファイルを作成
cp apps/backend/.env.example apps/backend/.env
# → apps/backend/.env を開いてWAONログイン情報・DB接続情報を入力

# Step 5: Prismaマイグレーション
cd apps/backend
pnpm prisma migrate dev --name init
pnpm prisma generate

# Step 6: バックエンド起動確認
pnpm dev
# → http://localhost:3001 で起動確認

# Step 7: フロントエンド起動確認（別ターミナル）
cd apps/frontend
pnpm dev
# → http://localhost:3000 で起動確認
```
